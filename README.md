<!-- GETTING STARTED -->
## Getting Started


### Prerequisites

Install package manager [yarn](https://classic.yarnpkg.com/en/docs/install/#mac-stable)

### Installation

2. Clone the repo
   ```sh
   git clone https://github.com/rhryansu/crwn-clothing.git
   ```
3. Install Yarn packages
   ```sh
   yarn install
   ```
